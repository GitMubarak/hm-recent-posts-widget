== Description ==

[HM Recent Posts Widget](http://www.hossnimubarak.com) is a simple plugin that will display the recent posts in your sidebar/widget panel area or in Post or Page with Ticker facility. You need to set featured image in post section to display the thumbnail. The post content is hidden in widget area or in mobile display, but shows in device which is larger than 767px. To display in post or page use the shortcode [hm_recent_posts]

**Plugin Features:**

* Easy to install
* Fully integrated in WordPress
* Integrated with ticker for both post/page or widget (New with V2.0)
* Ticker direction option available (Up/Down) (New with V2.0)
* Number of posts to display
* Option for post category selection
* Post title color option (New with V2.0)
* Responsive
And more to come...

== Installation ==

Install HM Recent Posts Widget as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search HM Recent Posts Widget
3. Click to install
4. Activate the plugin
5. Set the widget in Admin Widgets area.

**Install by upload plugin:**

1. Download the latest version of HM Recent Posts Widget (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded hm-recent-posts-widget.zip and click on "Install Now" button
6. Activate the plugin
7. HM Recent Posts will appear in Admin Widget area.

== Frequently Asked Questions ==

= Post thumbnail not displaying =

You need to set featured image in post section to display the thumbnail.

= How to show in Widget/Sidebar? =

Go to Appearance > Widgets. Find HM Recent Posts. Drag and drop in your widget area.

= How to show in Post/Page? =

Just use the shortcode: [hm_recent_posts]
